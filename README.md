### Set Up

1. Install virtualenv
```shell
pip install virualenv
virtualenv alaskaEnv
```
2. Activate virtualenv
```shell
source download.sh
```
3. Install dependencies
```shell
pip install -r requirements.txt
```
4. Run Python Install.py to download libraries
```shell
python install.py
```
4. Run Model
```shell
flask run
```
